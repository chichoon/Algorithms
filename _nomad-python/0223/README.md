## 직접 함수 만들기
- def funcname():
- 그리고 함수 내용은 indent (공백 4개) 를 주고 안쪽에 작성
- 한번 만들어두면 무한정 사용가능

## 함수에 argument (인자) 넣기
- def funcname(arg1, arg2, ...):
- 인자에 기본값을 주는 방법
    - def funcname(arg1 = 'default', arg2 = 'default')
    - 함수 사용 시에 인자를 제대로 넣지 않아도 default 값이 들어가므로 제대로 실행됨
